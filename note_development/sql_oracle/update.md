# UPDATE

*使用update遇到的bug*



1. update set后更新多条数据时，直接用逗号分隔，==不要用and==

+ 正确的

```sql
UPDATE "QUESTIONNAIRE_TEST" 
SET PDFURL = NULL , STATUS = 0 
WHERE
	TEST_INVOKE_KEY  	
	IN ( SELECT TEST_INVOKE_KEY FROM "QUESTIONNAIRE_TEST" WHERE PDFURL LIKE 'D:%' )
```

+ 错误的

```sql
UPDATE "QUESTIONNAIRE_TEST" 
SET PDFURL = NULL , AND STATUS = 0 
WHERE
	TEST_INVOKE_KEY 
	IN ( SELECT TEST_INVOKE_KEY FROM "QUESTIONNAIRE_TEST" WHERE PDFURL LIKE 'D:%' )
```

+ 错误的

```sql
UPDATE "QUESTIONNAIRE_TEST" 
SET PDFURL = NULL AND STATUS = 0 
WHERE
	TEST_INVOKE_KEY  
	IN ( SELECT TEST_INVOKE_KEY FROM "QUESTIONNAIRE_TEST" WHERE PDFURL LIKE 'D:%' )
```
